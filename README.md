# snake-game
一个专为移动端优化的、具备 iOS 审美 的经典贪吃蛇网页游戏。

项目亮点
为了解决网页版贪吃蛇在手机端“手感不佳”和“操作延迟”的痛点，本作进行了多项底层优化：
• iOS 视觉风格：采用 Apple 设计规范，包含深色模式背景、大圆角矩形按钮、以及毛玻璃视觉反馈。
• 穿墙模式 (Edge Wrapping)：取消了撞墙死亡的挫败机制，让游戏过程更加解压。
• 指令队列优化：引入输入缓冲逻辑，解决了转弯时的“不跟手”感，确保指令执行丝滑。
• 网格辅助线：内置极简像素网格，帮助玩家精确预判蛇头与食物的坐标对齐。
• 强力点击反馈：按钮具备动态缩放与颜色高亮反馈，确保护理体验中的每一次点击都有迹可循。

技术实现
• 核心架构：纯 HTML5 + CSS3 + Vanilla JavaScript。
• 渲染引擎：Canvas API 绘制 60FPS 丝滑动画。
• 响应式适配：支持全型号 iPhone/Android 屏幕自适应。

玩法操作
1.手机端：点击下方的虚拟方向键进行控制。
2.电脑端：使用键盘方向键 (↑ ↓ ← →) 进行控制。
3.规则：蛇可以穿过边界从另一侧出现，唯一的结束条件是撞到自己的身体。
